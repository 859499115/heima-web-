### 2.while 循环--**不明确循环次数**

 语法:

```js
while (循环条件) {
    重复执行代码(循环体)
}
```

注意事项:

1.变量起始值

2.终止条件

3.变量变化量

例子:

```js
let i = 1//变量起始值
while(i <=10){
    console.log(`输出是${i}`)//输出1到10数字
    i++//变量变化量
}
```

### 3.循环退出

 1)continue--结束本次循环,继续下次循环

 2)break--退出循环
