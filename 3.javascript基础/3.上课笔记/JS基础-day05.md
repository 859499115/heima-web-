# 内置对象

### 1.语法:

~~~js
for(声明记录循环次数的变量; 循环条件; 变化值){
    循环体
}
例子:
//输出1到10
for(let i = 1; i<=10; i++){
console.log(i)
}
~~~

### 2.最大价值是遍历数组arr

~~~js
let arr =['关羽', '张飞', '赵云', '黄忠', '马超']
for (let i= 0; i <=arr.length-1;i++){
    //arr[0]指的是'关羽'
    console.log(arr[i])//打印五个人名,数组遍历从0开始查找
}
~~~

## **Array数组**--按顺序保存数据的数据类型

里面可以放任何数据类型

### 1.语法:

~~~js
let 数组名 =[数据,数据2,数据3,.......]
例子:
let arr =['小明','熊王','大白','我的']
~~~

### 3.数组查删增改

~~~js
1.查
arr[索引号]
2.改"重新赋值"
arr[索引号]=新值

arr.push(新内容)--数组后面添加

arr.shift()--删除第一个值//()里面没有值
~~~

